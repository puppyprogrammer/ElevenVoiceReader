# Privacy Policy for ElevenVoiceReader

## Last Updated: February 2026

This Privacy Policy describes how ElevenVoiceReader ("we", "us", or "our") collects, uses, and protects information when you use our Chrome extension.

## Information We Collect

### Data Stored Locally
ElevenVoiceReader stores the following information locally on your device:
- **ElevenLabs API Key**: Encrypted and stored securely for text-to-speech functionality
- **Voice Preferences**: Your selected voice and audio settings
- **Volume and Speed Settings**: Your preferred playback configurations
- **Extension Settings**: UI preferences and configuration options

### Data Transmitted to Third Parties
- **Text Content**: Selected text is sent to ElevenLabs API servers for speech synthesis
- **API Authentication**: Your ElevenLabs API key is used to authenticate requests
- **Usage Analytics**: Basic extension usage data (if enabled in future versions)

## How We Use Information

### Primary Purpose
- **Text-to-Speech Conversion**: Selected text is processed by ElevenLabs to generate audio
- **Settings Persistence**: Your preferences are saved locally for consistent experience
- **Extension Functionality**: API keys enable voice synthesis features

### Third-Party Processing
- All text-to-speech processing occurs on ElevenLabs' secure servers
- We do not store or analyze the content of text you process
- ElevenLabs' privacy policy applies to TTS processing

## Data Security

- **Local Storage**: All sensitive data is stored locally using Chrome's secure storage APIs
- **Encryption**: API keys are handled securely and never transmitted unnecessarily
- **No Data Collection**: We do not collect personal information, browsing history, or usage analytics
- **Third-Party Security**: ElevenLabs uses industry-standard security measures

## Data Sharing

We do **not** share, sell, or rent your personal information to third parties, except as necessary for extension functionality:
- **ElevenLabs API**: Required for text-to-speech conversion
- **No Advertising**: We do not use your data for advertising purposes
- **No Tracking**: We do not track your browsing behavior or website visits

## Your Rights

- **Data Control**: You can delete all stored data by removing the extension
- **API Key Management**: You can change or revoke your ElevenLabs API key anytime
- **Data Deletion**: Uninstalling the extension removes all locally stored data

## Third-Party Services
